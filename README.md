# Socket IO Client
#### This is a Realtime Client application built using ReactJS and uses Socket.io
#### By [Paul Gichuki](https://paulgichuki.netlify.com/)
## Description
This is a Realtime Client app built using ReactJS and uses Socket.io to receive realtime communications from a Realtime Socket.io server. Any time a new record is updated on the server, the client immediately updates
## Setup/Installation Requirements
#### Requirements: Node
* Clone the repository
```
    $ git clone https://github.com/PaulinhoNdegwa/Socket-IO-Client.git
```
* Install dependencies  
```
    $ npm install OR yarn install
```
* Run the server
```
    $ npm start
```
